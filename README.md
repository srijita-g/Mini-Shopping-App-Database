
# 🛒 Mini Amazon Project

A mini e-commerce application using MySQL and HTML/CSS/JS that simulates Amazon-like product, user, and order management.

## 🚀 Features
- Product listing with price, brand, and rating
- User roles (Admin / Customer)
- Order and payment details
- Reviews system

## 💻 Tech Stack
- HTML, CSS, JavaScript
- MySQL

## ▶️ How to Run
- Open `index.html` in browser (or use Live Server)
- Run SQL script in MySQL Workbench

## 📌 Note
Frontend uses static data. Backend integration can be added later.
